-- APP_URL/move?x=0&y=0&direction=N&command=LMLMLMLMM

This endpoint accepts get parameters as x and y values for coordination, direction value (N,W,E,S) and command value to
 move.
 
 L means "Turn left"
 R means "Turn Right"
 M means "Move Forward"
